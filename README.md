# HR Employee Attrition Analysis Using Python

## 📌 Project Overview

This project analyzes employee attrition data using Python to identify the key factors affecting employee turnover and provide business insights to improve employee retention.

## 🎯 Objectives

* Analyze employee attrition patterns.
* Identify factors affecting employee turnover.
* Explore relationships between employee characteristics and attrition.
* Generate actionable business insights.

## 🛠️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Analysis Performed

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Statistical Analysis
* Business Insights Generation

## 📈 Key Insights

* Employees with lower job satisfaction had higher attrition rates.
* Monthly income significantly impacted employee retention.
* Employees with fewer years at the company were more likely to leave.
* Work-life balance played an important role in employee attrition.

## 💡 Business Recommendations

* Improve employee engagement programs.
* Enhance career development opportunities.
* Review compensation and benefits policies.
* Focus on work-life balance initiatives.
